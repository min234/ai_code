# ai-code

AI 기반 코드 분석 도구입니다.  
프로젝트의 소스 코드를 분석하여 **언어 변환, 리팩토링, 의존성 점검**을 도와줍니다.

이 도구는 “자동으로 다 고쳐주는 마법”보다는  
**개발자가 판단할 수 있는 근거를 제공하는 보조 도구**를 목표로 합니다.

---

## 주요 기능

### 1. 코드 분석 & 리팩토링
- 프로젝트 내 여러 파일을 분석
- 코드 구조 개선 제안
- 언어 변환(예: Python → TypeScript 등) 지원

### 2. 의존성 분석
- `requirements.txt`, `requirements.in`, `package.json` 등 의존성 설정 파일 자동 판별
- 사용되지 않는(unused) 의존성 탐지
- 누락된(missing) 의존성 탐지
- 버전 충돌(conflict) 및 위험한 범위 분석
- 과도하게 오래된(outdated) 패키지 경고

### 3. 안전한 적용 흐름
- 실제 파일 수정 전 **diff 미리보기 제공**
- 사용자 확인 후에만 파일 적용
- 의존성 파일 포맷 유지(`-r`, `-e`, marker 등)

> ⚠️ 이 도구는 의존성을 “자동 설치”하지 않습니다.  
> 파일 수정 이후 실제 설치는 사용자가 직접 `pip install` / `npm install`을 수행해야 합니다.
